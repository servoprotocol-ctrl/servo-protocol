    10 // QUESTIONS

    FAQ

    Who controls a machine's money?

    Its owner. Always. The machine can only spend inside the rules the owner set — a daily cap, an approved list — and the owner can freeze it or pull the money out at any time.

    What if a robot gets hacked?

    The thief gets one day's allowance at most, and only where the approved list points. Then the owner revokes the key and binds a new one. The blast radius is pocket change, not a bank account.

    How do AI agents pay?

    With one web request. An agent hits a paid service, gets back a price it can read, pays from its account, and gets the goods. The whole dance takes seconds. (It uses HTTP 402 — an old web status code called "Payment Required" — finally doing its job.)

    Is this a real RWA?

    The most real kind. A robot exists in the world, does real work, and earns real money. Servo makes that income provable, then turns it into shares people can own. Most RWAs just sit there. Ours work.

    Can I actually buy shares today?

    Yes. Offerings are live in the app, and a real buyer already owns 10% of our first station. Shares pay out in USDG, automatically, as the machine earns.

    Do robots really own stock?

    One does. Our delivery bot earned a dividend on its charging trades and claimed it as tokenized NVDA — a real swap, on a real market. It's small, it's real, and it's onchain where you can check it.

    Is the code open?

    Yes — Apache 2.0, built in public, with 85 passing tests. Every contract is verified on Blockscout so anyone can read exactly what the money does.
